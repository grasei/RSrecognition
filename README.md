# 🎙️ Диктовка✨

Скрипт на Python для голосового ввода текста на **русском** языке. Позволяет диктовать текст и приостанавливать запись в нужный момент. 

## 🚀 Основные возможности
- **Голосовой ввод**: Преобразование речи в текст в реальном времени.
- **Ручное управление**: Запись можно приостановить в любой момент.
- **Удобная индикация**: Индикация записи отображается значком около курсора.


## 🤖 О проекте и разработке
Этот проект был спроектирован и реализован при участии ИИ-помощников **chat.z.ai** и **Google AI**. Код оптимизирован для работы в среде Windows 11.

## 📥 Как запустить 
Для работы программы требуется pytnon 3.12. При его наличии достаточно выполнить два шага:

1. **Скачать скрипт**: Просто скачайте файл `ruspeech.py` в отдельную папку.
2. **Установить библиотеки**: Откройте терминал и установите все зависимости одной командой:
   
pip install   numpy  sounddevice keyboard  pyperclip  onnx_asr 
 
При первом распознавании будет выдано предупреждение и будет cкачана модель gigaam-v3-e2e-rnnt. Эта [модель](https://github.com/salute-developers/GigaAM/blob/main/README_ru.md) предобучена на 700 тысячах часах русской речи.

Если вы не хотите ставить pytnon и библиотеки, вы можете воспользоваться [готовой сборкой](https://github.com/grasei/RSrecognition/releases/latest). Смотрите  раздел по установке из готовой сборки ниже.


## Клавиши управления
1. **Запись и прекращение записи**: Print Screen, если не используется для захвата экрана.  В этом случае в параметрах системы Параметры — Специальные возможности — Клавиатура опция «Используйте клавишу PRINT SCREEN для открытия захвата» должна отключена.  В противном случае используется Ctrl+Win.
2. **Приостановка записи**: Правый Ctrl.

## Журналирование
Выполняемые действия протоколируются в окне терминала. Поэтому любую распознанную фразу можно скопировать в буфер обмена, выделив её и нажав клавишу Enter.

## Индикация под курсором
🔴 -- идёт запись звука  
⏸️ -- запись приостановлена  
⏳ -- обработка записи



## Удаление
1. pip uninstall   numpy  sounddevice keyboard  pyperclip  onnx_asr 
2. Удалить файл `ruspeech.py`
3. Удалить модель из папки C:\Users\<Имя_пользователя>\.cache\huggingface\hub

## Установка из готовой сборки
Скачайте zip архив с именем вида GigaAM_Voice_Dictation_v1.x_Win64.zip.
Распакуйте его в любое место (желательно не на рабочий стол, а в корень диска, чтобы избежать длинных путей).
Найдите внутри файл ruspeech.exe и запустите его.
Важно: При первом запуске программа скачает модель (~1.2 ГБ), дождитесь завершения в консоли.
Из-за специфики сборки с PyInstaller некоторые антивирусы могут выдать предупреждение. Добавьте папку с программой в исключения, если это произошло.
Убедитесь, что в настройках конфиденциальности Windows разрешен доступ приложений к микрофону.
При первом старте программа скачивает веса модели GigaAM-v3 (~1.2 ГБ) в папку пользователя (.cache/huggingface).
